Box Model Properties

Property Description Example Values
width, height Content area dimensions 200px, 50%, auto
padding Space inside border 10px, 5% 10px, 1em 2em 3em 4em
border Border around padding 1px solid black, 5px dotted #ccc
margin Space outside border 20px, 10px auto, 1rem 2rem
box-sizing How dimensions are calculated content-box, border-box
